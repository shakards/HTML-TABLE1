Table Caption

Table captions in HTML are used to add a descriptive title to your tables. Styling the table caption
with CSS helps improve the visual hierarchy and make your tables easier to understand.
This tutorial will show you how to style table captions effectively.

Example of Styled Table Caption

Employee Details
Name Age Location
John 30 New York
Mary 25 Los Angeles

HTML and CSS Code:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Table Caption Styling</title>
  <style>
    table {
      width: 60%;
      border-collapse: collapse;
    }
    table, th, td {
      border: 2px solid black;
    }
    th, td {
      padding: 10px;
      text-align: left;
    }
    th {
      background-color: #007bff;
      color: white;
    }
    caption {
      caption-side: top;
      font-weight: bold;
      font-size: 1.2em;
      padding: 10px;
      background-color: #d63384;
      color: white;
    }
  </style>
</head>
<body>
  <table>
    <caption>Employee Details</caption>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>Location</th>
    </tr>
    <tr>
      <td>John</td>
      <td>30</td>
      <td>New York</td>
    </tr>
    <tr>
      <td>Mary</td>
      <td>25</td>
      <td>Los Angeles</td>
    </tr>
  </table>
</body>
</html>
