Table with Borders and Styling


In HTML, tables are used to display data in a structured format, with rows and columns.
Styling tables, especially adding borders, makes the data more readable and visually appealing.
In this article, we will cover how to add borders to HTML tables, as well as some additional styling
techniques to enhance the table's appearance.

Adding Borders to a Table

Borders can be added to tables using the border property in CSS.
You can apply borders to the entire table, specific rows, headers, and
data cells. For example, to add a border around the entire table and its cells,

And This is An Example:


Name Age Location
John 30 New York
Mary 25 Los Angeles

HTML and CSS Code:

          <!DOCTYPE html>
          <html lang="en">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Table with Borders</title>
            <style>
              table {
                width: 60%;
                border-collapse: collapse;
              }
              table, th, td {
                border: 2px solid black;
              }
              th, td {
                padding: 10px;
                text-align: left;
              }
              th {
                background-color: #f2f2f2;
              }
            </style>
          </head>
          <body>
            <table>
              <tr>
                <th>Name</th>
                <th>Age</th>
                <th>Location</th>
              </tr>
              <tr>
                <td>John</td>
                <td>30</td>
                <td>New York</td>
              </tr>
              <tr>
                <td>Mary</td>
                <td>25</td>
                <td>Los Angeles</td>
              </tr>
            </table>
          </body>
          </html>

Stylng The Table

In HTML, tables are a fundamental way to display structured data in rows and columns.
However, plain tables can be difficult to read or visually unattractive. Styling tables with
CSS enhances readability and aesthetics, making them an essential tool for presenting
data in a clear, user-friendly format.

    Responsive Tables Example

    This example demonstrates how to make tables responsive, ensuring readability on small screens.

    Employee Age Department Location
    John 30 Sales New York
    Mary 28 Marketing San Francisco
    Steve 32 HR Chicago

    HTML and CSS Code:

    <!DOCTYPE html>
<html>
<head>
  <style>
    /* Basic table styling */
    table.table-responsive {
      width: 60%;
      border-collapse: collapse;
      margin: 20px 0;
      font-size: 1em;
      min-width: 300px; /* Minimum width for the table */
    }

    th, td {
      padding: 12px 15px;
      text-align: left;
    }

    th {
      background-color: #4CAF50;
      color: white;
    }

    tr:nth-child(even) {
      background-color: #f2f2f2;
    }

    /* Responsive styling for smaller screens */
    @media screen and (max-width: 600px) {
      table.table-responsive {
        display: block;
        overflow-x: auto; /* Enable horizontal scrolling */
        white-space: nowrap; /* Prevent text wrapping */
      }

      table.table-responsive thead {
        display: none; /* Hide the header on small screens */
      }

      table.table-responsive tr {
        display: block;
        margin-bottom: 10px;
      }

      table.table-responsive td {
        display: block;
        text-align: right;
        font-size: 1.2em;
        position: relative;
        padding-left: 50%;
      }

      table.table-responsive td::before {
        content: attr(data-label);
        position: absolute;
        left: 0;
        width: 50%;
        padding-left: 15px;
        font-weight: bold;
        text-align: left;
      }
    }
  </style>
</head>
<body>
  <table class="table-responsive">
    <thead>
      <tr>
        <th>Employee</th>
        <th>Age</th>
        <th>Department</th>
        <th>Location</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td data-label="Employee">John</td>
        <td data-label="Age">30</td>
        <td data-label="Department">Sales</td>
        <td data-label="Location">New York</td>
      </tr>
      <tr>
        <td data-label="Employee">Mary</td>
        <td data-label="Age">28</td>
        <td data-label="Department">Marketing</td>
        <td data-label="Location">San Francisco</td>
      </tr>
      <tr>
        <td data-label="Employee">Steve</td>
        <td data-label="Age">32</td>
        <td data-label="Department">HR</td>
        <td data-label="Location">Chicago</td>
      </tr>
    </tbody>
  </table>
</body>
</html>
