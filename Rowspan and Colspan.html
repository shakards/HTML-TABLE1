Table Rowspan and Colspan

The rowspan and colspan attributes in HTML tables allow you to merge cells either vertically
or horizontally. This tutorial demonstrates both features in separate tables.

1. Table Rowspan

Employee Age Department
John 30 Sales
30 Marketing

HTML and CSS Code for Rowspan:

<!DOCTYPE html>
<html>
<head>
  <style>
    table {
      width: 60%;
      border-collapse: collapse;
    }
    table, th, td {
      border: 2px solid black;
    }
    th, td {
      padding: 10px;
      text-align: left;
    }
    th {
      background-color: #007bff;
      color: white;
    }
    td[rowspan] {
      background-color: #d63384;
      color: white;
      text-align: center;
    }
  </style>
</head>
<body>
  <table>
    <tr>
      <th>Employee</th>
      <th>Age</th>
      <th>Department</th>
    </tr>
    <tr>
      <td rowspan="2">John</td>
      <td>30</td>
      <td>Sales</td>
    </tr>
    <tr>
      <td>30</td>
      <td>Marketing</td>
    </tr>
  </table>
</body>
</html>

2. Table Colspan

Employee Age Department
Mary Human Resources
Steve 28 Finance

HTML and CSS Code for Colspan:

<!DOCTYPE html>
<html>
<head>
  <style>
    table {
      width: 60%;
      border-collapse: collapse;
    }
    table, th, td {
      border: 2px solid black;
    }
    th, td {
      padding: 10px;
      text-align: left;
    }
    th {
      background-color: #007bff;
      color: white;
    }
    td[colspan] {
      background-color: #6c757d;
      color: white;
      text-align: center;
    }
  </style>
</head>
<body>
  <table>
    <tr>
      <th>Employee</th>
      <th>Age</th>
      <th>Department</th>
    </tr>
    <tr>
      <td>Mary</td>
      <td colspan="2">Human Resources</td>
    </tr>
    <tr>
      <td>Steve</td>
      <td>28</td>
      <td>Finance</td>
    </tr>
  </table>
</body>
</html>
